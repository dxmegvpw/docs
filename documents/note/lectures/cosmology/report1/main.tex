設問(1)\\

フリードマン方程式
\begin{equation}
  H^2
  =
  \frac{8\pi G}{3}\rho
  -
  \frac{Kc^2}{a^2}
  +
  \Lambda
\end{equation}
に$\rho=3M/4\pi a^3$を代入すれば
\begin{equation}
  \left(  
    \frac{\dot{a}}{a}
  \right)^2
  =
  \frac{2GM}{a^3}
  -
  \frac{Kc^2}{a^2}
  +
  \Lambda  
  \label{eqn:1-freedman}
\end{equation}
である。これを$Kc^2$について解けば
\begin{equation}
  Kc^2
  =
  a^2
  \left\{
    \frac{2GM}{a^3}
    -
    \left( \frac{\dot{a}}{a} \right)^2
    +
    \Lambda
  \right\}
  \label{eqn:1-1}
\end{equation}
となる。

フリードマン方程式\eqref{eqn:1-freedman}を微分すれば
\begin{equation}
  \frac{2a^2\dot{a}\ddot{a}-2a\dot{a}^3}{a^4}
  =
  -\frac{6GM}{a^4}+\frac{2Kc^2}{a^3}
\end{equation}
であり、これに\eqref{eqn:1-1}を代入して整理すると
\begin{equation}
  a^2\dot{a}\ddot{a}
  -
  a\dot{a}^3
  +
  a\dot{a}^2
  +
  GM
  -
  a^3\Lambda
  =
  0
\end{equation}
となる。\\


設問(2)\\

$a=a^{\ast}=\text{const.}$なら、$\dot{a}=0$なので、フリードマン方程式\eqref{eqn:1-1}と前問の結果は、それぞれ
\begin{gather}
  \frac{Kc^2}{{a^{\ast}}^2}
  =
  \frac{2GM}{{a^{\ast}}^3}
  +
  \Lambda
  \\
  GM
  -
  {a^{\ast}}^3\Lambda
  =
  0
\end{gather}
となり、$G$を消去して整理すれば
\begin{equation}
  {a^{\ast}}^2
  =
  \frac{Kc^2}{\Lambda}
  >
  0
\end{equation}
である。したがって、$K>0$であり、$a^{\ast}$は
\begin{equation}
  {a^{\ast}}
  =
  \sqrt{\frac{Kc^2}{\Lambda}}
\end{equation}
となる。\\
