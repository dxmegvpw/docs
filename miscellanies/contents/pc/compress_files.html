How to compress folders individually using 7zip

Sometimes I want to compress several folders individually. I think you will naively select all the target folders and compress them using 7zip or another compressor application, but you will find that it does not work as expected. If we want to achieve our goals, some tricks are needed.

Main contents

First of all, 7zip needs to be installed. The installer can be downloaded here. Once on the site, you should get the installer for the appropriate Operating System.

After downloading, make compress_7zip.bat and type contents of the bat file as follows:

 @echo off

 for %%f in (%*) do (
  echo %%f
  "C:\Program Files\7-Zip\7z.exe" a -mmt=on -mx=9 %%~dpf%%~nf.zip %%f
 )

 pause

[compress_7zip.bat]

  • Both -mnt=on and -mx=9 are options.
  • -mnt=on enable us to multithreading.
  • -mx=9 is a compression ratio settings.

All you have to do then is drag all the folders you want to compress into the .bat file. 7zip will start immediately and create the zip files.
