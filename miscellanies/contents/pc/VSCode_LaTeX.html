My setup for VS Code and LaTeX

When I write notes or report assignments, I usually use LaTeX since it produces beautifully formatted mathematical equations, and above all, it is enjoyable to use.

To make our LaTeX lives happier, I usually use Visual Studio Code (VS Code) for a text editor. It helps us to write TeX code and introduce some features, like Grammarly or git etc., more easily. I cannot imagine my life without VS Code for me.

I will show you one of the installation and setup examples for LaTeX and VS Code. Our strategies are as follows:

  1. Install the Windows Subsystem for Linux (WSL) and VS Code. We will use a virtual environment and activate LaTeX on it.
  2. Log in to the VS Code account and synchronies the settings. (If you do not have a Microsoft account, you should create one as it will be easier to set up next time). Install TeX Live on Linux from the command prompt.
